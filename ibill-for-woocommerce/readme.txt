iBill.my payment gateway plugin for WooCommerce. 

== Description ==

iBill.my payment gateway plugin for WooCommerce. This plugin enable online payment using online banking (for Malaysian banks only). Currently iBill.my is only available for businesses that reside in Malaysia.

== Installation ==

1. Make sure that you already have WooCommerce plugin installed and activated.
2. From your Wordpress admin dashboard, go to menu 'Plugins' and 'Add New'.
3. Key in 'iBill.my' in the 'Search Plugins' field and press enter.
4. It will display the plugin and press intall.
5. Activate the plugin through the 'Plugins' screen in WordPress.
6. Go to menu WooCommerce, settings, Checkout, iBill.my and fill in your merchant id and secret key. You can retrieve the merchant id and secret key from iBill.my merchant site at https://ibill.my/merchant/?ng=client. 
7. Make sure the 'Enable this payment gateway' is ticked. Click on 'Save changes' button.
8. In iBill.my Api section make sure you key in your return URL as http://your_domain/ .

== Frequently Asked Questions ==

= Do I need to sign up with iBill.my in order to use this plugin? =

Yes, we require info such as merchant id and secret key that is only available after you sign up with iBill.my.

= Can I use this plugin without using WooCommerce? =

No.

= What currency does it support? =

Currently iBill.my only support Malaysian Ringgit (RM).

= What if I have some other question related to iBill.my? =

Please open a ticket by email to info@ibill.my and look for menu support.
